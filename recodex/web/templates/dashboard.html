Welcome to RecodeX

RecodeX is a modern media transcoding service that automatically processes video files from configured watch folders.

Getting Started

  1. Click the Configuration button above to access the settings panel
  2. Add transcoding profiles to define how your videos should be processed
  3. Configure watch folders to monitor directories for new video files
  4. Start the full service with recodex start to begin automatic transcoding

Quick Setup

Use the configuration interface to:

  • Create custom transcoding profiles
  • Add folders to monitor for video files
  • Adjust settings like file extensions, output paths, and more
